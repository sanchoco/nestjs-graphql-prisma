# Nest.js, graphQL, prisma example code

## Installation

```bash
$ npm i -g yarn
$ yarn
$ npx prisma generate
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```


## Prisma
```bash
# database show data
npx prisma studio
# database sync
npx prisma db push
```
